# 概要
「Arduino/i71」は、
int71こと「[整数71](https://www.youtube.com/@int71)」が「PSコントローラUSB変換」等を自作するために作成した、
「[Arduino](https://www.arduino.cc/)」向け手順、各種リソース群です。

# 登録されているファイルについて
- 手順書「HowtoInstall(J).txt」
- 環境カスタマイズ用設定ファイル
- ブートローダ不使用、メモリ節約のための、置き換え用標準ライブラリ、空ブートローダイメージ
